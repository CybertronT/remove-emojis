# Remove Emojis

## Description
Remove emojis from your code to make it cleaner. Works on any file type (HTML, JS, Python, Markdown, etc.).  

## How to use
1. Open any file in VS Code.  
2. Press `Ctrl+Shift+P` (or `Cmd+Shift+P` on Mac).  
3. Type **Remove Emojis from Code** and run the command.  
4. All emojis in the file will be removed instantly.  


## Features
- Remove all emojis from code and text files.
- Works with all VS Code-supported file types.
- Simple, one-command workflow.
